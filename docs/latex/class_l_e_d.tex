\doxysubsection{Detailed Description}
Clase para gestionar un diodo \doxylink{class_l_e_d}{LED} conectado a un pin GPIO. 

Permite realizar operaciones básicas como encender, apagar, alternar el estado o realizar un parpadeo (brillar).
