\doxysubsection{Detailed Description}
Gestiona la lógica de empaquetado y emisión de datos de sensores. 


\begin{DoxyItemize}
\item Organiza las lecturas en el formato estándar de i\+Beacon, utilizando el campo \textquotesingle{}Major\textquotesingle{} para el ID de la medida y un contador, y el campo \textquotesingle{}Minor\textquotesingle{} para el valor de la medición. 
\end{DoxyItemize}
