Detailed Description

Gestiona la lógica de empaquetado y emisión de datos de sensores.

  • Organiza las lecturas en el formato estándar de iBeacon, utilizando el campo 'Major' para el ID de la medida y un contador, y el campo 'Minor' para el valor de la medición.
