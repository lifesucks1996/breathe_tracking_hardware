Detailed Description

Clase para la gestión de sensores analógicos y simulados.

Author
Rocio
Date
11/11/2025

Este archivo contiene la lógica para calcular concentraciones de O3 mediante un sensor electroquímico, así como la estimación del nivel de batería y la generación de datos simulados para CO2 y Temperatura.
